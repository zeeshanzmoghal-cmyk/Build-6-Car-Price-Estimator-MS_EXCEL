# Car Price Estimator

## Overview
Car Price Estimator is an Excel-based tool designed to estimate the value of a Toyota vehicle based on key factors such as age, usage, condition, model, and modifications.

The goal of this project is to simulate real-world vehicle depreciation and customization effects in a structured spreadsheet model.

## Features
- Price degredation per the age of vehicle
- Price degredation per years of ownership
- Price difference per trims and models
- Price degredation per crash
- Price change per added part/tech
- Price change for modified vehicles

## How it works
Users input vehicle details into the designated cells (Row 3). The spreadsheet processes the inputs using predefined formulas to generate a final estimated price.

If invalid or incomplete data is entered, the calculator will not return a result.

## Tools used
- Microsoft Excel
- Formulas and conditional logic
